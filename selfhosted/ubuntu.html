Intro

I recently started using Ubuntu, due to my old computer crashing. This is a list of useful things I've learned about it and its ecosystem of tools since I've started using it. I probably skipped a few things I already knew, and probably missed a few things I don't know yet.

Good Things

  • Midori

Midori is a lightweight browser. It's not particularly fast, but it's fairly configurable (e.g. no-script, no-image, etc. options native), and it's memory lightweight. This is really useful for loading StackOverflow and email. It's based on WebKit (so...Safari).

  • AutoKey

I use a lot of hotkeys (not ratpoison level, but getting there slowly). AutoKey allows for full scripts to be bound to keys, including sending keys back to specific windows. This means that in, for example, Google Chrome, you can remap keys to specific pages, or complex actions. As a sidenote, Google Chrome should really get custom key settings.
